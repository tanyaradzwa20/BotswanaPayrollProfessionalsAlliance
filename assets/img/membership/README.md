# Membership Images

Please add the following images to this folder:

1. **trainee-membership.jpg** - Image for Payroll Trainee Membership card
   - Recommended size: 800x500 pixels
   - Should represent students/graduates/learning

2. **professional-membership.jpg** - Image for Payroll Professionals Membership card
   - Recommended size: 800x500 pixels
   - Should represent experienced professionals/business

3. **benefits.jpg** - Image for Additional Benefits card
   - Recommended size: 800x500 pixels
   - Should represent networking/collaboration/growth

## Image Guidelines:
- Format: JPG or PNG
- Recommended dimensions: 800x500 pixels (16:10 ratio)
- File size: Keep under 500KB for optimal loading
- Style: Professional, high-quality images that represent payroll professionals
- Content: Avoid stock photos with watermarks

## Current Status:
All cards currently use a fallback placeholder image. The cards will automatically display your images once you add them to this folder with the correct filenames.
